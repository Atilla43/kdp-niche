## Детализация Затрат

### Keepa API (€19/мес = ~$21/мес)

| Параметр | Значение |
|----------|----------|
| Лимит запросов | 60/минуту |
| Токены | Базовое количество |
| Что даёт | BSR история, цена, отзывы |
| Чего НЕ даёт | Search volume |

**Скрытые затраты:** При активном использовании токены заканчиваются быстрее. Реальная стоимость может быть €30-50/мес.

**Решение:** Агрессивное кэширование снижает потребление на 70-80%.

### Keywords Everywhere ($21/год)

| Параметр | Значение |
|----------|----------|
| Credits | 100,000/год |
| 1 credit | 1 keyword lookup |
| Срок действия | 1 год с покупки |

**Расчёт:** 100K credits = ~500-1000 полных анализов ниш (если каждая ниша = 100-200 keywords)

### Vercel Free Tier

| Ограничение | Значение | Проблема |
|-------------|----------|----------|
| Function timeout | **10 секунд** | Keepa может отвечать дольше |
| Bandwidth | 100GB/мес | Достаточно |
| Invocations | 150K/мес | Достаточно |

**Решение:** Кэширование API-ответов в Supabase.

### Supabase Free Tier

| Ограничение | Значение | Проблема |
|-------------|----------|----------|
| Storage | 500MB | Достаточно |
| **Пауза после 7 дней** | Критично! | Проект выключится |
| MAU | 50,000 | Достаточно |

**Решение:** GitHub Actions ping каждые 6 дней (бесплатно).

### Stripe

| Комиссия | Значение |
|----------|----------|
| Базовая | 2.9% + $0.30 |
| С Stripe Tax | 3.4% + $0.30 |
| **Итого на $19** | ~$1.00-1.20 (5-6%) |

**Дополнительно:**
- Refund: комиссия НЕ возвращается
- Chargeback: $15 за каждый

## Ценообразование

| Тариф | Цена | Целевая аудитория |
|-------|------|-------------------|
| **Free** | $0 | Новички, пробуют |
| **Pro** | $19/мес или $149/год | Активные издатели |

**Почему $19/мес:**
- Publisher Rocket: $97 разово (= 5 мес нашей подписки)
- Book Bolt: $10-20/мес
- Психологический порог: < $20 = "дёшево"

## Unit Economics

| Метрика | Значение | Комментарий |
|---------|----------|-------------|
