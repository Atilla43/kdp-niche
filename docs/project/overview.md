# KDP Niche Finder

> Онлайн-платформа для автоматического поиска прибыльных ниш на Amazon KDP

## Что это?

SaaS-платформа, которая анализирует Amazon KDP и помогает авторам находить ниши с высоким спросом и низкой конкуренцией. Использует данные BSR (Best Seller Rank) и объём поиска ключевых слов для расчёта **Niche Score** — единой метрики прибыльности ниши.

## Для кого?

| Сегмент | Боль | Как помогаем |
|---------|------|--------------|
| **Начинающие KDP-авторы** | Не знают как искать ниши, тратят недели на ручной анализ | Автоматический поиск + понятный Niche Score |
| **Опытные издатели** | Публикуют много книг, нужна автоматизация | Batch-анализ, алерты на новые ниши |
| **Дизайнеры low-content** | Ищут trending темы | Топ ниш недели, тренды |
| **Фрилансеры и агентства** | Делают книги на заказ | Экспорт данных, отчёты для клиентов |

**География:** США, UK, Германия (основные рынки KDP)
**Язык платформы:** English

## Какую проблему решает?

| Проблема | Решение |
|----------|---------|
| Ручной поиск ниш занимает часы/дни | Автоматический скрининг тысяч категорий за минуты |
| Непонятно, есть ли спрос на нишу | Search volume через Keywords Everywhere + BSR данные |
| Сложно оценить конкуренцию | Анализ отзывов, рейтингов, возраста книг в топе |
| Нет понимания трендов | Исторические данные и алерты на растущие ниши |
| Дорогие существующие решения ($97+) | Freemium-модель: бесплатный старт, $19/мес Pro |

## Niche Score — Ключевая Метрика

```
Niche Score (0-100) = Demand Score + Competition Score + Opportunity Score
```

| Компонент | Что измеряет | Источник данных |
|-----------|--------------|-----------------|
| **Demand Score** | Спрос на нишу | BSR топ-10 книг (Keepa) + Search Volume (Keywords Everywhere) |
| **Competition Score** | Сложность входа | Количество отзывов, рейтинги конкурентов |
| **Opportunity Score** | "Окно" для входа | Новые книги в топе, разброс BSR, цены |

## Функционал MVP

### Free тариф (5 поисков/мес)
- Базовый поиск ниш
- Niche Score
- BSR, отзывы, цены топ-книг

### Pro тариф ($19/мес)
- Безлимитный поиск
- Search volume ключевых слов
- Исторические тренды
- Алерты на новые ниши
- Сохранение и экспорт (CSV)
- Email-дайджест топ ниш недели

### Чего НЕ будет
- Создание книг / контента
- Публикация на KDP
- Дизайн обложек
- Написание текстов
