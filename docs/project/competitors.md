## Детальный Анализ

### Publisher Rocket (Kindlepreneur)

**Плюсы:**
- Огромная база пользователей и доверие
- Много данных, точные показатели
- Разовая покупка = нет recurring payments
- Сильный бренд (Dave Chesson)

**Минусы:**
- Устаревший десктопный интерфейс
- Нет веб-версии
- Разовая покупка = нет регулярных обновлений
- $97 — высокий порог входа для новичков

**Наше преимущество:**
- Веб-платформа (доступ отовсюду)
- Бесплатный старт
- Современный UI

### Book Bolt

**Плюсы:**
- Комбинация: исследование + дизайн
- Доступная цена ($9.99/мес)
- Хорошо для low-content книг

**Минусы:**
- Перегруженный интерфейс
- Фокус только на low-content
- Медленная работа

**Наше преимущество:**
- Фокус на одной задаче (поиск ниш)
- Чистый интерфейс
- Все типы книг

### KDP Spy

**Плюсы:**
- Дёшево ($9.97/мес)
- Простое Chrome расширение

**Минусы:**
- Только Chrome extension
- Ограниченный функционал
- Нет комплексного анализа

**Наше преимущество:**
- Полноценная платформа
- Niche Score вместо сырых данных

### Helium 10

**Плюсы:**
- Мощная аналитика
- Множество инструментов
- Большая база данных

**Минусы:**
- Для Amazon FBA, не для KDP
- Дорого ($29-$229/мес)
- Overk ill для авторов

**Наше преимущество:**
- Заточено именно под KDP
- Доступная цена

## Наши Дифференциаторы

### 1. Freemium модель
- Бесплатный старт снижает барьер
- Publisher Rocket требует $97 сразу
- Book Bolt — только 3-дневный trial

### 2. Niche Score
- Единая метрика вместо десятков цифр
- Понятно даже новичкам
- Учитывает demand + competition + opportunity

### 3. Современный веб-интерфейс
- Доступ с любого устройства
- Чистый UI (shadcn/ui)
- Быстрая работа

### 4. Фокус на одной задаче
- Только поиск ниш
- Нет лишних функций
- Простота использования

## Слабые Места Конкурентов

### Publisher Rocket
- Нет веб-версии в 2025 году — анахронизм
- Разовая покупка = нет мотивации обновлять
- UI из 2015 года

### Book Bolt
- Слишком много функций
- Медленный интерфейс
- Только low-content

### KDP Spy
- Только расширение браузера
- Минимум аналитики
- Нет комплексного скоринга
