## Почему Включён

Helium 10 — **НЕ прямой конкурент**, но часто упоминается как альтернатива для Amazon KDP авторов из-за инструмента **Cerebro**.

> "Helium 10 is primarily for Amazon FBA, but it has a feature called Cerebro that is useful for Amazon KDP."

---

## Для Книг: Cerebro

### Что Это

Cerebro — инструмент reverse ASIN lookup, который позволяет:
- Найти ключевые слова конкурентов по ASIN
- Анализировать keyword performance
- Получить данные о search volume

### Как Используют Авторы

1. Берут ASIN успешной книги
2. Запускают Cerebro
3. Получают список keywords, по которым книга ранжируется
4. Используют для своих книг

## Почему Авторы Выбирают Helium 10

1. **Уже используют для FBA** — продают и товары, и книги
2. **Хотят advanced keyword data** — больше данных чем Publisher Rocket
3. **Готовы платить** — серьёзные издатели с бюджетом

---

## Почему Авторы НЕ Выбирают Helium 10

1. **Только книги** — не нужны FBA функции
2. **Бюджет** — $29+/мес vs $199 one-time (Publisher Rocket)
3. **Сложность** — слишком много функций
4. **Нет специфики** — нет категорий, BSR analysis, niche score

## Вывод

Helium 10 — **не угроза** для нашего продукта:
- Разная целевая аудитория
- Разный ценовой сегмент
- Разный фокус

Это **overkill** для большинства авторов, особенно:
- Новичков
- Low-content creators
- Авторов с ограниченным бюджетом
